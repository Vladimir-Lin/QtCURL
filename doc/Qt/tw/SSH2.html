編譯libSSH2











原始碼網站：https://www.libssh2.org
目前版本：1.7.0
最後更新日期：2016/04/04






























Linux 32 / 64

copy openssl include and lib into a directory like the following:

> openssl
  +- include
  +- lib

copy zlib include and lib into a directory like the following:

> zlib
 +- include
 +- lib

./configure --disable-shared --enable-static --with-libssl-prefix=/wherever/openssl --with-libz-prefix=/wherever/zlib
make


Mac OS X Clang 64

./configure --disable-shared --enable-static


Windows 32 / 64

Windows 32

Look for libssh2-1.5.0/win32, there is a libssh2 VC project file.  Open it and compile, that's all.


Windows 64

Add a x64 in VS and compile it.








cmake -G "Visual Studio 12 Win64" ..\libssh2-1.7.0 -DBUILD_SHARED_LIBS=1 -DCMAKE_INSTALL_PREFIX=H:\libssh2\x64\shared\release -DOPENSSL_ROOT_DIR=G:\Qt\3rdparty\openssl\OpenSSL\shared\x64\release





Android ARMv7

copy openssl and zlib like Linux.

use one step compilation script : ssh2-android.sh


iPhone ARMv7

copy openssl include and lib into a directory like the following:

> openssl
  +- include
    +- openssl
  +- lib

copy zlib include and lib into a directory like the following:

> zlib
 +- include
 +- lib

use one step compilation script : ssh2-ios.sh
